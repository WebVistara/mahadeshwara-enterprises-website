# 🏢 Mahadeshwara Enterprises Website

A modern and responsive multi-page website developed for **Mahadeshwara Enterprises** to showcase their services, portfolio, and company information with a professional design.

---

## 🚀 Features

* 🌐 Multi-page website (Home, About, Services, Clients, Gallery, Contact)
* 📱 Fully responsive design for all devices
* ⚡ Fast loading with optimized images (`.webp`, `.avif`)
* 🧾 Clean and professional business UI
* 📷 Gallery section to display projects
* 📞 Contact page for customer inquiries
